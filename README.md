# 🚀 Luyện LeetCode

Repository này dùng để lưu lại quá trình luyện bài tập trên LeetCode.

## 🎯 Mục tiêu

* Nâng cao tư duy giải thuật và cấu trúc dữ liệu
* Chuẩn bị cho phỏng vấn kỹ thuật
* Rèn luyện kỹ năng code và tối ưu thuật toán

## 📚 Nội dung

* Các bài LeetCode từ dễ → khó
* Phân loại theo chủ đề:

  * Array / String
  * Linked List
  * Stack / Queue
  * Tree / Graph
  * Dynamic Programming

## 🛠️ Ngôn ngữ sử dụng

* Java (chính)
* Có thể thêm C++ / Python nếu cần

## 📈 Tiến độ

* Cập nhật thường xuyên theo từng ngày / tuần

## 🔥 Ghi chú

Mỗi bài đều cố gắng:

* Viết code rõ ràng
* Tối ưu độ phức tạp
* Ghi lại cách tư duy
